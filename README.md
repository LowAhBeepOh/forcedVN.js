# forcedVN.js
A visual novel game engine for JavaScript (HTML)
Inspired by RenPy.

## What is forcedVN.js?
Visual Novel game engine.

## Why is forcedVN.js?
Since it was made.

## Why was forcedVN.js made, and why is it called that?
My friend forced me to make a visual novel game with him, so I called it ForcedVN since it's a forced project and since it's a visual novel (VN) thing.

## How easy is it to make a game?
Should be pretty easy; you mostly just touch the Labels thing inside the engine code.

## Full Command List
